League of Legends Stats ↗

2018 | ReactJS, Redux, HTML, CSS, MongoDB, Node.js, ExpressJS, Heroku

As part of learning ReactJS and Javascript I have built a knowledge platform for League of Legends (League of Legends is a highly competitive PC game). Where players can improve their skills by staying data informed. This App offers the following:

  • › Player match history, it includes very detailed information such as history of all the items the player bought during every match. Death count, kill count, hero skill leveling order etc.
  • › 'Most played with section' that shows how well the player did with their most played friends (top 5).
  • › Guides on how to play every single heroes in the League of Legends game, including videos on hero skills, best items, strongest and weaknes opponents etc.

Note: Many of the features no longer working with real data due to the recent League of Legends API end-point changes. I plan to update the App with the latest API end points as well refactor the app with latest ReactJS.

To use the app with legacy data that I stored prior to the API changes, please switch the top-right search region to 'NA' and search for Caenos. This should demonstrate how the player page look like. Alternatively, feel free to use the Champions Tab to see all the guides and stats for each hero.
